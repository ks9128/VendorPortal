# Vendor Portal

A full-stack platform for vendors to manage products and receive public feedback.

## Features
- **Vendor Dashboard**: Manage profile, add/edit products, track stats.
- **Product Management**: Upload product images (Cloudinary) and set pricing.
- **Public Profiles**: Shareable links for vendors with reviews.
- **Responsive Design**: Fast and mobile-friendly UI.

## Tech Stack
- **Frontend**: React, Vite, TailwindCSS.
- **Backend**: Node.js, Express, MongoDB.
- **Deployment**: Vercel (Frontend & Backend).

## Live Demo
- **Frontend**: https://vendor-portal-zeta.vercel.app/
- **Backend API**: https://vendor-portal-backend.vercel.app/

## Setup
1. Clone the repo.
2. Install dependencies: `npm install` in both `frontend` and `backend` folders.
3. Setup `.env` files (see example).
4. Run `npm run dev`.
